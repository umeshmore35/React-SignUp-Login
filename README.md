## Get Started

![alt text](https://media.giphy.com/media/11BbGyhVmk4iLS/giphy.gif)


## Clone project

```
git clone https://github.com/umeshmore45/React-LogIn.git
```
## Install Dependency

```
npm install
```
## Run App
```
npm run start
```
